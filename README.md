# 🔗 URL Shortener

Um encurtador de URLs desenvolvido com FastAPI, SQLAlchemy e PostgreSQL.

A aplicação permite criar links curtos, redirecionar usuários e acompanhar estatísticas básicas de acesso através de uma interface web simples e intuitiva.

🚀 **Aplicação Online:** https://encurtador-url-ymvj.onrender.com/

📚 **Documentação da API:** https://encurtador-url-ymvj.onrender.com/docs

## 🏗 Arquitetura

O projeto foi estruturado utilizando separação de responsabilidades:

* API → Rotas HTTP
* Service → Regras de negócio
* Repository → Persistência de dados
* Models → Entidades SQLAlchemy
* Schemas → Validação Pydantic

Essa organização facilita manutenção, testes e escalabilidade da aplicação.

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de desenvolvimento back-end, incluindo:

* APIs REST
* Arquitetura em camadas
* Banco de dados relacionais
* ORM com SQLAlchemy
* Testes automatizados
* Deploy em produção
* Integração Front-end e Back-end
